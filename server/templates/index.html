This is the FiBar diameter measuring tool.

Insert a scanning electron microscopy (SEM) iage (accepted PNG/JPG/TIF format, max 2MB). The iages are expected to contain fibers, stay in magnification between 500 and 20000 and have a scale in the lower left corner of the image.
It is also expected for the uploaded file name to include the respective magnification used, as this enables to extract fibers from the image correctly. All of the constraints of this tool are provided here. Some example images are provided below.


Link to Package Github repository

The FiBar tool is actively improved and updated. This tool also has a bacteria counting pipeline which is under development.
The tool has been created in collaboration with the Kogermann Lab at the Institute of Pharmacy of the University of Tartu.
